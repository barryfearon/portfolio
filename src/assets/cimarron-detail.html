Cimarron Casino

A static website design produced for the Cimarron Casino as a portal to the iOS & Android download centres for the native apps. The Casino site included instruction on how to use the App, utilising graphics and text in an FAQ format to assist players that may have had issues when accessing the games.

Built using HTML 5, CSS3, Bootstrap 4.5 and JQuery the site was a multipage site that was easy to navigate and fast to load.
