GreySnow Group

This static website design was produced for the GreySnow Group as a B2B platform that would inform and promote the business activity for the group.

Built using HTML 5, CSS3, Bootstrap 4.5 and JQuery the site was a one page scrolling site that was easy to navigate and fast to load. Using lazyload techniques to improve speed. Animation was also employed to introduce movement into the titles sections
