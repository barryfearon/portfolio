Digital Design Online Portfolio

Native Gaming

US online gaming company Native Gaming requested a new front end for its existing business to allow for a better User Experience to it's online offering for Casino and Online Poker.

The site was deigned using Figma to prototype the screens which were then signed off by the client and then prepared for production.

The website is a fully responsive site built using HTML5, CSS3 / SASS stylesheets, Bootstrap 4.5 front end tool kit and the NextJS React framework.
